# Arrow function

> ## **Definição**

É uma forma simplificada de declarar funções.

É considerado uma _function expression_.

> ## **Exemplos**

### **Arrow function com 1 linha**

```js
const falaOi = () => console.log("Oi");

falaOi();
```

### **Arrow function com múltiplas linhas (bloco)**

```js
const falaOi = () => {
  console.log("Oi");
  console.log("Olá");
};

falaOi();

/*
Oi
Olá
*/
```
