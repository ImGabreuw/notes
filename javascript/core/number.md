# Tipo de dado `number`

> ## Números inteiros

### **Definição**

São números sem casas decimais.

### **Exemplo**

```js
console.log(100) // 100
```

---

> ## Números flutuantes

### **Definição**

São números com casas decimais (com pontos flutuantes, na programação)

## **Exemplo**

```js
console.log(3.1415) // 3.1415
```
