# Tag `<i>`
  
> ## **Definição**

A tag `<i>` serve para mudar apenas o estilo da letra para itálico.
  
> ## **Exemplo**

### **Código**:
   
```html
<p>Lorem <i>ipsum dolor</i> sit amet consectetur, adipisicing elit. Dolorum, labore.</p>
```
    
### **Demonstração**:
    
<p>Lorem <i>ipsum dolor</i> sit amet consectetur, adipisicing elit. Dolorum, labore.</p>
