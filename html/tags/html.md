# Tag `<html>`

> ## **Definição**

A tag `<html>` é obrigatória em qualquer arquivo `.html`.

Todas tags no HTML são filhas dela.

Engloba todas as tags do documento.

> ## **Atributos**

* `lang`: especificar o idioma para o navegador

  ```html
  <html lang="pt-BR">
    ...
  </html>
  ```

> ## **Exemplo**

```html
<html>
  ...
</html>
```
