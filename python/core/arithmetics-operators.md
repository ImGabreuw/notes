# Operadores aritméticos

|             Nome              | Operador |   Código    |   Resultado   |
| :---------------------------: | :------: | :---------: | :-----------: |
|            Adição             |   `+`    |  `10 + 10`  |     `10`      |
|           Subtração           |   `-`    |  `10 - 5`   |      `5`      |
|         Multiplicação         |   `*`    |  `10 * 10`  |     `100`     |
|            Divisão            |   `/`    | `10 / 2.2`  | `4.545454...` |
| Divisão truncada (apenas int) |   `//`   |  `10 // 3`  |      `3`      |
| Divisão truncada (com float)  |   `//`   | `10 // 2.2` |     `4.0`     |
|         Exponenciação         |   `**`   |  `2 ** 10`  |    `1024`     |
|            Módulo             |   `%`    |  `10 % 3`   |      `1`      |
