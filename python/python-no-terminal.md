# Python no terminal

### Terminal do Python

* **Entrar no terminal**: `$ python`

* **Sair do terminal**: `>>> exit` ou `CTRL + Z`

### Comandos

* **Visualizar a versão do Python**: `$ python --version`

* **Executar um arquivo `.py`**

  * **Sintaxe**: `$ python <caminho até arquivo>`

  * **Exemplo**: `$ python ./main.py`
